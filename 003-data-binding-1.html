Data binding

An example of associating an array with html elements using D3.js data binding.

List of all countries: 


  let countries = [
    'Argentina', 'Australia', 'Austria', 'Bangladesh', 'Belgium',
    'Brunei', 'Cameroon', 'Chile', 'China', 'Cuba', 'Denmark',
    'Egypt', 'Eritrea', 'Estonia',
  ]

  // we are going to append to #main-text
  d3.select('#main-text')
    // possibly the most confusing part of D3.js, this creates an li
    // these are the elements we want to associate with the data
    // but obviously they do not exist at the moment
    .selectAll('li')
    // provide the data
    .data(countries)
    // here is where we link the li elements with the data
    .enter()
    // create and append the li elements
    .append('li')
    // set the text for the li elements to be the data
    // d is automatically set to be the array entry
    .text( (d) => d)
